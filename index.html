About this website


This website was built responsively using HTML5, CSS, JS and some Jquery magic. The Parallax Scrolling effect was made with Jquery and not with plugins like Stellar.js or Skrollr.js because i had to figure something out for a contribution at the FH St. Pölten. Fonts are served through Google Fonts.


As a current project I'm building this website completely new with Slim, SCSS, JS and Jquery too. I also want to try out some of the Parallax Scrolling plugins. Hopefully the code behind this site will be alot cleaner and smoother after all. ;)


If you want to get in touch with me, feel free to write me on any platform from the {contact} me section down there.
